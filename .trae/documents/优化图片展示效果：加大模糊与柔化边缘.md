针对您反馈的“图片周围过渡生硬”的问题，您希望在不改变背景（模糊填充）的前提下，让前景图片与背景的融合更加柔和。

我建议采用 **“边缘羽化（Vignette）”** 或 **“内阴影遮罩”** 的方案来柔化前景图的硬边：

1.  **添加遮罩层**：在前景图上方添加一个绝对定位的 `div`。
2.  **内阴影（Inset Shadow）**：给这个 `div` 设置一个四周向内的阴影（或渐变），颜色可以是黑色/深灰色（低透明度）或者直接使用 CSS 的 `mask-image` 属性来实现边缘的透明渐变。
    *   **方案 A (推荐)**：使用 `box-shadow: inset 0 0 20px rgba(0,0,0,0.1)`。这会给图片四周加一圈淡淡的阴影，消除硬切边的感觉，让图片像是“嵌”在背景里一样。
    *   **方案 B (更柔和)**：使用径向渐变背景遮罩，使图片边缘逐渐变透明，与模糊背景融合。

考虑到实现的便捷性和兼容性，以及“不想改变背景”的需求，**方案 A（内阴影）** 是最稳妥且效果明显的选择。它能给前景图增加一种“悬浮感”或“嵌入感”，消除生硬的边界线。

**修改计划：**

1.  修改 `front/src/pages/Discover.vue`。
2.  在前景 `<img>` 标签的同级（或外层容器内），添加一个新的 `div` 覆盖层。
3.  设置该覆盖层为 `absolute inset-0 pointer-events-none`（确保不影响点击）。
4.  添加内阴影样式，例如 `shadow-[inset_0_0_20px_rgba(0,0,0,0.1)]` 或类似的 Tailwind 类。
5.  或者，直接给前景图本身添加一个轻微的 `drop-shadow`，但这可能会让它看起来与背景脱离，内阴影融合感更好。
