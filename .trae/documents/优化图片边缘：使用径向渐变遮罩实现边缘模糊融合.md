您反馈的问题是：虽然添加了内阴影，但图片本身的边缘依然比较硬，尤其是在浅色背景上。您希望前景图片的边缘能够**模糊化**，从而真正地“融化”到模糊的背景中，而不是仅仅加一个黑框阴影。

为了实现**图片周边模糊并与背景融合**的效果，最有效的方法是使用 CSS 的 `mask-image` (或 `-webkit-mask-image`) 属性，结合径向渐变 (Radial Gradient)。

**解决方案：**

1.  **移除**之前的内阴影 `div`（或者保留作为辅助，视效果而定，建议移除以避免干扰）。
2.  **应用遮罩**：直接对前景图片应用一个径向渐变遮罩。
    *   遮罩中心部分（图片主体）是不透明的（可见）。
    *   遮罩边缘部分逐渐变为透明（不可见）。
    *   由于底层是模糊的背景图，当上层图片的边缘变透明时，就会透出底下的模糊背景，从而产生完美的“融合”效果。

**CSS 实现逻辑：**
```css
mask-image: radial-gradient(circle, black 70%, transparent 100%);
-webkit-mask-image: radial-gradient(circle, black 70%, transparent 100%);
```
这意味着图片中心 70% 的区域是完全清晰的，从 70% 到 100%（边缘）逐渐变得透明，露出底下的模糊层。

**修改计划：**

1.  修改 `front/src/pages/Discover.vue`。
2.  找到前景 `<img>` 标签。
3.  添加行内样式或 Tailwind 类来实现 `mask-image`。
    *   由于 Tailwind 默认没有直接的径向渐变遮罩类，我们可以使用 `style` 绑定或者添加自定义类，或者利用 Tailwind 的 `mask-image` 插件（如果项目没装，就用内联样式最稳妥）。
    *   我们将使用内联样式：`style="-webkit-mask-image: radial-gradient(circle, black 60%, transparent 100%); mask-image: radial-gradient(circle, black 60%, transparent 100%);"`。
4.  同时，为了让效果更好，可以适当调整底层模糊背景的透明度和亮度，确保过渡自然。

这将彻底解决“生硬”的问题，因为图片边缘不再是直角切断，而是渐隐消失在背景中。
